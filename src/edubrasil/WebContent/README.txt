Para Executar Localmente
	Para utilizar a aplicação em alguns browsers se faz necessários algumas mudanças, caso você use um dos navegadores listados abaixo realize as mudanças
	antes de abrir o index.html, caso contrário apenas abra o index.html no navegador.
	-Chrome:
			1º No ícone do navegador clique com o botão direito do mouse e selecione Propriedades
			2º Nas Propriedades do Google Chrome mude adicione o seguinte código: --allow-file-access-from-files
				O Destino deve ficar da seguinte maneira: "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe"--allow-file-access-from-files
			3º Clique em Aplicar (Talvez o Windows requisite aprovação de administrador para confirmar a alteração)
			4º Clique em ok
			5º Abrar o index.html no navegador.

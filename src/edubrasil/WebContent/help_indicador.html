Como é feita a analise estatistica dos indicadores? A analise foi realizada após um estudo dos dados. Os dados paraibanos seguem um comportamento de uma Distribuição Normal*, a partir disso podemos identificar valores dentro desses dados que não seguem a normalidade do que seria aceitavél para os valores dos indicadores. Da forma como estão os dados, sabemos que aproximadamente 70% deles estão dentro da normalidade, fora desse intervalo temos outros valores que ja merecem atenção, eles são destacados em níveis, baseado no Desvio Padrão* (ou ౮). Temos também intervalos especiais, ou seja, não representam valores ruins, e sim bons, eles são os dados que estão agrupados a partir do 1౮, 2౮ e 3౮.
